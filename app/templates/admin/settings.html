How to configure OpenRouter

  1. Go to openrouter.ai and create an account.
  2. Navigate to Keys and click Create key.
  3. Copy the key — it starts with sk-or-v1-.
  4. Paste it above and click Save & test. The app will verify the key works immediately.
  5. Alternatively, set OPENROUTER_API_KEY in Railway's Variables tab — this takes precedence over the database key and is more secure.
